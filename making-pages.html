How do i make the content on the page?

lets start with making text and modifying its basic looks first, for every page you ever make, put <!DOCTYPE html>
<html>
<head></head>
<body></body>
</html> in the a file of any name (readme.html), the editor you use is unimportant, hell even the built in editor from github is perfectly acceptable to be doing this all that matters is that you can write text into the body element of the page and page meta into the head element

Whats an element?

an element is <tagName></tagName> (some elements, meta and br specifically of note, can be expressed with <tagName/> or <tagName> instead) common tag names to use are h# (# being any number) for headers and a href="#url" for hyperlinks (#url being the url to goto when clicked, the href="#url" should NOT go in the second half of the element)

I have a page now... but its kinda ugly

no need to worry! this is just how your browser defaults the display for your content (very bland, very untextured) to edit the looks of the content in the body element, you will need css! (normally inside the head, but not explicitly required that it is) to write css simply put a <style></style> and the inside of the style element will be your css
css works by matching some set of elements and then defining what visual properties those matched elements should have, for now we will be handling basic element matching, where you match by tag and nothing else
to start, getting any styling done at all, inside the style element you added put body {
font: sans-serif;
}, this will make your webpages font change to the browsers sans-serif selection, rather then purely being default only
from here you can say, center the text, or change the background color, for now i will only list some bare minimums but dont be worried about understanding css, as 99% of any webdev is just "google how i do this, copy there implementation" and nothing further in thought, so for the most part just do that
to change text color simply set the color attribute in the css to what ever color you want (just google "color picker" if you dont have any that will give hex colors) (if you confused on what "setting a css attribute is", its the same thng as the "font: sans-serif;" thing we just added, just with font replaced with attribute name and sans-serif replaced with the attribute value, the attributes can be placed anywhere inside the curly braces and even be duplicated multiple times, with the last one being the only one to take efect)
to set the background color then, much like text color, set the background-color attribute to a color, but theres more! gradiants are also valid colors, being defined like linear-gradient(to bottom, #colorone 0px, #colortwo 100vh) with #colorone replaced with the color you want at the top and #colortwo as the bottom color
to make all text centered simply set the text-align attribute to center
dont be worried about it looking bad to others as the point is to share yourself safely in a way you enjoy, not explicitly to give pleasure or content to others
